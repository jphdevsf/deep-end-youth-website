# Deep End Youth Website

This repository contains the source code for the official website of the band **Deep End Youth**. The site is a single-page app featuring streaming and purchase links, a lyrics carousel, and an animated background. It is built with **Next.js** as a static export and deployed to **Netlify**.

---

## Project Overview

- The site is a static single-page application generated by Next.js (`output: 'export'`).
- The build outputs static HTML, CSS, JavaScript, and image assets into the `/out` folder.
- The resulting files are served via **Netlify** at [www.deependyouth.com](https://www.deependyouth.com).
- Google Analytics and Google Tag Manager are integrated for tracking.

---

## Tech Stack

- **Next.js 15** — React framework with static export support
- **React 19** — UI library
- **TypeScript** — Type-safe JavaScript
- **Tailwind CSS 3** — Utility-first CSS framework
- **PostCSS** with **Autoprefixer** — CSS processing
- **ESLint** — Code linting with flat config

---

## Features

- **Album Card** — Displays album artwork with links to streaming services (Spotify, Apple Music, Amazon Music, YouTube, Pandora) and purchase platforms (Bandcamp, Amazon).
- **Lyrics Carousel** — Interactive song selection with lyrics display for all five EP tracks.
- **Animated Background** — Blurred album art background with a continuous hue-rotation animation.
- **Responsive Design** — Mobile-first layout using Tailwind CSS utility classes.
- **Google Fonts** — Josefin Sans loaded via `next/font` for optimal performance.

## Setup and Installation

### Prerequisites
- **Node.js** (LTS recommended: v20.x or later)
- **NPM**

### Install Dependencies
```bash
npm install
```

### Development
```bash
npm run dev
```
This starts the Next.js dev server at `http://localhost:3000`.

### Build
```bash
npm run build
```
This generates the static export into the `/out` folder.

### Lint
```bash
npm run lint
```

---

## Deployment

The site is deployed to **Netlify** via the configuration in `netlify.toml`:

- **Build command:** `npm run build`
- **Publish directory:** `out`
- Static assets (`/images/*`, `/_next/static/*`) are cached with a 1-year max-age.
- All routes are redirected to `index.html` for single-page app support.

---

## ESLint Configuration

The project uses ESLint with a flat config (`eslint.config.mjs`). Key rules:

- **2 spaces** for indentation
- **Single quotes** for strings
- **No semicolons**
